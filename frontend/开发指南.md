# 前端开发指南

Guideflow 的 Web 前端：面向 NCCN B 细胞淋巴瘤（默认 DLBCL 范围）的指南问答界面。原生 HTML / CSS / JS，无构建步骤。

## 文件结构

| 文件 | 说明 |
|------|------|
| `index.html` | 主界面（两栏布局 + 鉴权弹窗） |
| `app.js` | 状态管理、API 调用、SSE 流式、渲染逻辑 |
| `styles.css` | 样式 |
| `share.html` | 公开分享只读页（`?token=`） |
| `vendor/marked.min.js` | Markdown 解析 |
| `vendor/purify.min.js` | HTML XSS 消毒 |
| `package.json` | 可选：`npm run dev` 等价于本地静态服务 |

## 本地启动

**后端（:8001）**

```bash
# 建议在项目 .venv 中运行
.\.venv\Scripts\python.exe -m uvicorn backend.api.server:app --reload --host 127.0.0.1 --port 8001
```

**前端（:5173）**

```bash
# 方式 A：Python 静态服务
python -m http.server 5173 -d frontend --bind 127.0.0.1

# 方式 B：在 frontend 目录用 npm 脚本
cd frontend && npm run dev
```

浏览器打开 http://127.0.0.1:5173/ 。

也可通过后端挂载访问 http://127.0.0.1:8001/app/ （`backend/api/server.py` 将 `frontend/` 挂到 `/app`）。

默认 API 地址为 `http://127.0.0.1:8001`，可在 `index.html` 或运行时通过 `window.API_BASE` 覆盖。分享页还支持 URL 参数 `?api=` 指定后端。

### 相关环境变量（后端）

| 变量 | 说明 |
|------|------|
| `DATABASE_URL` | 默认 `sqlite:///data/app.db` |
| `AUTH_SECRET` | 会话 HMAC 密钥（**生产必设**；未设时开发环境会写入 `data/.auth_secret`） |
| `AUTH_ALLOW_PASSWORD_RESET` | 默认 `true`：允许邮箱直重置密码（本地演示）；生产请设 `0` |
| `CORS_ORIGINS` | 逗号分隔白名单，默认含 `127.0.0.1:5173` |
| `COOKIE_SECURE` | 生产 HTTPS 设 `true` |
| `QWEN_API_KEY` / `VLM_API_KEY` | 模型密钥（仅服务端，勿注入前端） |

## 功能清单

### 布局与导航

- **两栏**：左侧栏 + 主对话区（已移除原右侧证据栏）
- 侧栏可收起 / 展开；状态写入 `localStorage`（`gf_sidebar_collapsed`）
- 「开启新对话」位于侧栏顶部
- 顶栏品牌区 + 隐蔽「⚙ 工具」菜单

### 左侧栏 · 历史

- 登录后从 `GET /api/conversations` 拉取列表
- 按 **今天 / 7 天内 / 30 天内 / 更早** 分组展示
- 点击条目打开会话；当前会话高亮
- 条目「⋯」：**重命名**（`PATCH`）、**删除**（`DELETE`）
- 未登录时提示「登录后可同步历史记录」
- 会话过期（401）时提示重新登录

### 空状态与快捷提问

- 无消息时展示引导文案 + **4 条示例问题**（可点击直接提问）
- 示例涵盖一线治疗、亚型、TP53、CNS prophylaxis 等

### 对话与输入

- **真流式**：`POST /api/ask` 带 `stream: true`，消费 SSE 事件 `token` / `final`（非整段假打字机）
- 非 SSE 响应时自动降级为一次性 JSON
- 请求携带 `conversation_id`（已登录且续聊时）、`trace: true`
- 生成中显示加载转圈；进行中可因新提问 `AbortController` 取消上一轮
- 输入框：**回车发送**、**Shift+回车换行**、高度随内容自适应（最高约 160px）
- 输入框可 **展开 / 收起**（展开固定约 180px）
- 发送后清空输入并收起展开状态

### 答案渲染

- Markdown（marked）+ DOMPurify 消毒
- 优先按 `answer_paragraphs` 分段；否则用 `answer_markdown`
- 流程图按 `anchor_paragraph` **内联**到对应段落；无锚点的图归入文末「相关流程图」
- 支持 **compact / full** 双视图；点击「放大（含脚注）」打开 lightbox
- 行内 `[Sn]` / `[Gn]` / 参考文献 `[n]` 渲染为可交互引用按钮
- 引用 **悬浮卡**：展示来源摘要；`See All` 滚动到文末 References；点击引用也可跳转
- 文末可折叠 **References**（合并 `sources` + `attached_references`，含 PMID / DOI 链接）

### 消息操作条

| 操作 | 行为 | 前置条件 |
|------|------|----------|
| 复制 | 复制 `answer_markdown` 纯文本，图标短暂变为 ✓ | 无 |
| 重说 | 取上一条用户问题重新提问，去掉末尾 assistant 消息 | 无 |
| 点赞 / 点踩 | `POST /api/messages/{id}/feedback`，可取消 | 需登录且消息已持久化（有 `id`） |
| 分享 | 创建分享 token，复制 `share.html?token=...&api=...` 链接 | 需登录且当前对话已保存到云端 |

### 调试工具（⚙）

对**最近一次完整回答**的 payload 打开抽屉：

- **Trace** — JSON 格式化 trace
- **证据来源** — `sources` 卡片列表
- **图谱路径** — `graph_triples` 三元组
- **流程图** — `figures` 缩略图（可放大）

### 鉴权

- 邮箱 **注册 / 登录 / 退出**（Cookie 会话，`credentials: 'include'`）
- 弹窗支持 **忘记密码** → `POST /api/auth/reset-password`（本地直重置，无邮件）
- 密码框 **显示 / 隐藏** 切换
- 点击遮罩关闭弹窗（仅在遮罩上按下并松开，避免拖选文字误关）
- 未登录可问答；登录后问答写入服务端会话并出现在历史

### 本地持久化（未登录 / 刷新）

| 存储 | 键 | 内容 |
|------|-----|------|
| `localStorage` | `gf_sidebar_collapsed` | 侧栏折叠 |
| `localStorage` | `gf_active_conversation_id` | 上次打开的会话 ID（登录后） |
| `sessionStorage` | `gf_chat_draft` | 当前页草稿：消息列表、`lastPayload`、临时 `conversationId` |

刷新后：已登录则优先恢复 `gf_active_conversation_id` 对应会话；否则尝试恢复 `sessionStorage` 草稿。

### 分享页（`share.html`）

- 只读展示：`GET /api/shared/{token}`
- URL：`share.html?token=...`，可选 `&api=` 覆盖后端地址
- 渲染用户消息 + AI Markdown 答案；无消息操作条
- 失效或缺少 token 时显示友好错误

## API 依赖（前端调用）

| 方法 | 路径 | 用途 |
|------|------|------|
| `POST` | `/api/ask` | 问答（SSE 或 JSON） |
| `GET` | `/api/auth/me` | 当前用户 |
| `POST` | `/api/auth/login` | 登录 |
| `POST` | `/api/auth/register` | 注册 |
| `POST` | `/api/auth/logout` | 退出 |
| `POST` | `/api/auth/reset-password` | 重置密码 |
| `GET` | `/api/conversations` | 历史列表 |
| `GET` | `/api/conversations/{id}` | 打开会话 |
| `PATCH` | `/api/conversations/{id}` | 重命名 |
| `DELETE` | `/api/conversations/{id}` | 删除 |
| `POST` | `/api/conversations/{id}/share` | 创建分享 |
| `POST` | `/api/messages/{id}/feedback` | 点赞 / 点踩 |
| `GET` | `/api/shared/{token}` | 分享只读 |
| `GET` | `/api/images/{filename}` | 页图 / 裁剪图（由 payload URL 引用） |

## 安全注意（上云）

1. 轮换 `.env` 中的模型 API Key（勿提交 git）
2. 设置强随机 `AUTH_SECRET`；生产关闭密码直重置（`AUTH_ALLOW_PASSWORD_RESET=0`），`COOKIE_SECURE=true`，收紧 `CORS_ORIGINS`
3. 建议反代同源托管静态页 + `/api`；SQLite 单 worker 或改 Postgres
4. 前端已对 Markdown 消毒；勿把密钥注入 `window.API_BASE` 以外的配置
